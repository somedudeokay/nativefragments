# Agent-Friendly Applications

> Native Fragments is designed so agents can build and inspect apps without reverse-engineering a transpiled bundle.

Agent-friendly applications are useful for people and agents:

- Server HTML is readable.
- Links are real anchors.
- Components are native Custom Elements.
- Runtime modules are browser JavaScript, not generated framework output.
- Route manifests make application shape explicit.
- `llms.txt` and Markdown docs give agents a stable entry point.

The same traits that help coding agents maintain an app also help browsing agents click, scrape, summarize, and test the rendered interface.
